# as-cinema-hall

This is a movie ticket booking website.

# About Project Technology:
* React JS (Frontend Framework)
* Bootstrap
* Express JS (Backend Framework)
* MongoDB (Database)
* Firebase (Hosting & google Authentication)
* Heroku (Server side deployment)
